Come si installa

CryptoSentinel si installa direttamente dall'APK. Bastano 4 passaggi.

Scarica l'APK

Tocca il pulsante "Scarica l'APK" qui sopra dal tuo telefono Android.

Apri il file

Apri il file scaricato dalle notifiche o dalla cartella Download.

Consenti l'installazione

Se richiesto, abilita "Installa da questa origine" per il browser.

Apri e via!

Installa, apri CryptoSentinel e inizia a monitorare il mercato.

Android potrebbe avvisarti perché l'app non arriva dal Play Store: è normale per le app distribuite via APK. Il codice è open source e verificabile su GitHub.
